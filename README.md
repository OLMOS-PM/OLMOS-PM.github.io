# Pablo M. Olmos — GitHub Pages starter

This folder is a static personal academic website prepared for GitHub Pages.

## How to publish

1. Create a GitHub account, if you do not already have one.
2. Create a new public repository named exactly:

   ```text
   YOUR-GITHUB-USERNAME.github.io
   ```

   For example, if your GitHub username is `pablomolmos`, the repository should be `pablomolmos.github.io`.

3. Upload all files in this folder to the root of that repository:

   ```text
   index.html
   assets/styles.css
   assets/pablo-yosemite.jpg
   .nojekyll
   ```

4. Go to the repository settings: `Settings → Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select branch `main` and folder `/root`, then save.
7. Your website should become available at:

   ```text
   https://YOUR-GITHUB-USERNAME.github.io/
   ```

## Updating the site

Edit `index.html` for text and links. Edit `assets/styles.css` for the visual design.

## Optional: custom domain

You can later point a custom domain to this site from `Settings → Pages → Custom domain`.
